Const createSignature

  • createSignature(name: string, signature: string | Blob): Promise<ISignature>
    • Defined in Documents/Signatures.ts:18

    Create a signature block. In a typical signing workflow, the user is asked at the beginning of the process to "adopt" a signature block to be used for all signature fields in the document. Thus, this is typically called one time to create and store a signature block. Thereafter, the ID of the signature block may be re-used for each signature field to be "stamped" by the user.

    • signature: string | Blob
